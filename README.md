# Setup
1. Clone or fork this repository.
1. Install yarn if needed:
    ```ps
    npm install -g yarn
    ```

1. Install dependencies:  
    ```ps
    yarn install
    ```

2. Start the React app:  
    ```ps
    yarn start
    ```
3. The React app will be running locally on localhost:3000/.
